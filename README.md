# 🐾 Clínica Veterinaria - Web App

Aplicación web para la administración de una clínica veterinaria, desarrollada con el stack MERN (MongoDB, Express, React, Node.js).

## ¿De qué trata?

Plataforma que permite gestionar una clínica veterinaria: registro de veterinarios, administración de pacientes (mascotas), control de citas y servicios.

## Tecnologías

- **Backend:** Node.js, Express.js, MongoDB, Mongoose
- **Frontend:** React *(en desarrollo)*

## Estado del proyecto

- ✅ Servidor y conexión a MongoDB
- ✅ Modelo y rutas de veterinarios
- 🔄 Autenticación con JWT
- ⏳ Frontend con React
- ⏳ Módulos de pacientes, citas y servicios
